# 💳 Previsão de Score de Crédito com Machine Learning

## 📌 Objetivo
Criar um modelo de Machine Learning capaz de prever o score de crédito de clientes com base em seus dados financeiros e comportamentais.

O modelo classifica os clientes em três categorias:

- Poor (ruim)
- Standard (ok)
- Good (bom)

Esse tipo de modelo pode ajudar bancos a avaliar risco de crédito de novos clientes.

---

## 🛠 Tecnologias Utilizadas

- Python
- Pandas
- Scikit-Learn
- Jupyter Notebook

---

## 📊 Etapas do Projeto

1. Importação e exploração dos dados
2. Tratamento de variáveis categóricas usando LabelEncoder
3. Separação dos dados em treino e teste (70/30)
4. Treinamento dos modelos de Machine Learning
5. Avaliação dos modelos com métricas de classificação
6. Previsão de score para novos clientes

---

## 🤖 Modelos Testados

- Random Forest
- KNN (K-Nearest Neighbors)

---

## 📈 Resultado

O modelo **Random Forest** apresentou melhor desempenho, com aproximadamente **83% de acurácia** na base de teste.
